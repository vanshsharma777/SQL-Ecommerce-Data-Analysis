# SQL E-Commerce Data Analysis

## Project Overview
This project demonstrates SQL data analysis using a sample E-Commerce database. It includes database creation, data insertion, SQL queries, joins, aggregate functions, and portfolio-level business insights.

## Database Tables
- Customer
- Product
- Orders
- Orders_Item
- Payments

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()
- INNER JOIN
- Aggregate Functions

## Business Questions Solved
- Total Customers
- Total Products
- Total Orders
- Total Revenue
- Delivered Orders
- Pending Orders
- Revenue by Payment Method
- Top Expensive Products
- Highest Order Amount
- Average Order Value
- Customer Order Details
- Product Quantity Sold
- Customer Payment Details
- Revenue by Customer
- Average Spending by Customer
